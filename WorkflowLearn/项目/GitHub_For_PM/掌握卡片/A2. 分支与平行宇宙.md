# 掌握卡片：A2. 分支与平行宇宙

## 🧠 我的理解
- **分支本质**：本质是**“指针”**（便利贴）。创建分支极其廉价，只是贴个标签，不复制文件。
- **隔离价值**：One Branch, One Feature。如果在一个分支里混做两件事（积分改版+登录文案），上线时就会被“捆绑销售”，导致风险耦合。

## 🔗 类比挂钩
**"便利贴指针"**
- 分支就是贴在 Commit 上的便利贴，标记谁是当前最新的进度。
- 移动分支（Reset）就是撕下便利贴贴到以前的 Commit 上。

## 🚫 踩过的坑
- **误区**：认为分支是整个文件夹的物理拷贝（Copy & Paste）。
- **真相**：分支只是指向某个 Commit 的轻量级动态指针。

## 📖 源真理定义
> **Branch (分支)**：指向某个 Commit 的可移动指针。它允许开发者在不影响主线的情况下进行并行开发。
