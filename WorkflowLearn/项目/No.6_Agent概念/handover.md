## 当前判断

这门课应被视为 `Agent / Workflow / Harness Engineering` 的核心课，不再只是 Agent 概念导论。

## 已冻结决策

1. 稳定主干只保留机制、边界、模式和判断线。
2. workflow 编排范式、DSL/schema、observability、恢复策略和 human-in-loop 进入主线。
3. 多 Agent 协作、Computer Use、具体框架和产品名只保留为时效层或案例层。
4. 课程输出必须能落到产品判断、方案设计和求职作品。
