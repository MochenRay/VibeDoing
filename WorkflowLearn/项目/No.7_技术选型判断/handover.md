# No.7_技术选型判断 交接页

> 最近确认时间：2026-06-01

## 当前判断

这门课应当成为 AI PM 的决策课，而不是单纯的技术方案对比课。

## 已冻结决策

1. 课程主线应覆盖 prompt / RAG / workflow / agent / fine-tune / multimodal / build-buy-assemble。
2. 成本、延迟、风险、维护、组织落地、ROI 必须和技术选型一起学。
3. 具体模型价格、榜单和短期性能快照只进时效层。
4. 输出必须能直接服务方案评审、立项和面试表达。

## 未决问题

1. 供应商锁定是否作为独立评审表，还是并入 `Build / Buy / Assemble`。
2. `Prompt / RAG / Workflow / Agent / Fine-tune` 的统一决策树是否单独沉淀为面试答题模板。
3. 多模态和 AI native surface 是否作为选型维度单列。
