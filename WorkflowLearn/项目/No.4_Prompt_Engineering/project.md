## 用途

本文件是 `No.4_Prompt_Engineering` 在新 `WorkflowLearnSystem` schema 下的项目首页。
当前项目已从旧的 Prompt 技巧集，重写为 `Prompt / Schema / Tools` 的正式课程。

## 目标

- 为 `AI 应用 PM + Agent / Workflow PM` 提供输入约束、结构化输出和工具合同的主课。
- 把 Prompt 从“调技巧”升级成“定义接口、定义约束、定义机器可执行合同”。
- 把 Generative UI / AI-native 交互纳入课程主线，而不是只把 AI 当聊天框。

## 成功标准

- `Prompt / Schema / Tools` 成为主线
- `Generative UI`、工具调用、结构化输出都有明确的产品化落点
- 稳定概念、时效项、验证任务和全景产出分离清楚
