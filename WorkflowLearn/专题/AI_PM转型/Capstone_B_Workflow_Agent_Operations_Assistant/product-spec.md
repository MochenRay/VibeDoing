# Product Spec

> 最近确认时间：2026-04-16
> 产品名：内部运营工单协同助手
> 类型：Workflow Agent Operations Assistant

## 目标用户

- 运营专员
- 客服运营经理
- 需要处理跨系统工单的支持团队

## 核心目标

- 减少人工切系统和复制粘贴
- 为低中风险工单提供可执行处理建议
- 对高风险工单自动整理 handoff 包给人工审批

## 核心功能

### 1. 工单理解与路由

- 识别工单类型
- 提取关键字段
- 选择需要调用的工具链

### 2. 跨系统信息整合

- 拉取 CRM 数据
- 拉取历史工单
- 拉取规则文档 / SOP

### 3. 处理建议生成

- 给出建议动作
- 给出所依据的数据或规则
- 标明不确定点

### 4. 人工接管

- 高风险或异常状态自动升级
- 输出摘要、证据、建议和未确定项

### 5. 运行可见性

- 展示当前步骤
- 展示工具调用结果
- 展示失败原因和接管原因

## 不做什么

- 不自动执行高风险写入
- 不绕过人工审批
- 不在无足够依据时给强结论
