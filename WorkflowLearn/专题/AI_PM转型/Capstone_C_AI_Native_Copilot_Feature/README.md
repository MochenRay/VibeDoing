## 主题

`AI-native Copilot Feature`

当前默认样例是：`客户研究与方案撰写 Copilot`。
它也可以平移成销售方案 Copilot、客户成功工作台、研究与写作增强界面等场景。

## 目标

用一套完整案例证明三件事：

- 你能在 chat、copilot、inline edit、canvas/workbench、voice/live/multimodal 中做 surface 取舍，而不只是默认聊天框
- 你能把 feedback telemetry 回流到 eval dataset，并和 fallback、launch gate 一起做进 feature
- 你能把体验、ROI、成本结构、rollout 和可上线性放在同一个产品讨论里
