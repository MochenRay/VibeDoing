# eval-plan

## 1. 离线评估

- `brief completeness`
  - 是否覆盖客户目标、约束、风险和下一步
- `evidence correctness`
  - 关键陈述能否追溯到可信来源
- `section usefulness`
  - 方案章节是否达到“可继续编辑”而不是“只能参考”
- `risk detection recall`
  - 是否能识别价格、合规和交付高风险内容

## 2. 在线指标

- `time to first usable draft`
- `accepted suggestion rate`
- `manual edit ratio`
- `unsupported claim rate`
- `review escalation rate`

## 3. 上线门槛

- 高风险段落的无证据率低于设定阈值
- 关键章节的可用草稿率达到试点目标
- 人工审核路径完整可用
- 明确回滚条件和监控面板

## 4. 红队样本

- 会议纪要与 CRM 信息冲突
- 上传材料缺页或过期
- 用户要求输出无证据承诺
- OCR 或转写错误导致事实漂移
