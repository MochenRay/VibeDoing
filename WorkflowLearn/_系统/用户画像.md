# 用户画像

> [!NOTE]
> 这是全局用户画像，跨项目共享。新项目无需重复询问。

- **用户身份**：超级个体 (Super Individual)
- **技术背景**：关注个人效率赋能，寻求将 AI 融入工作流以提升产出质量与速度。
- **创建日期**：2026-01-09
- **最后更新**：2026-01-09

---

## 兴趣图谱

### 1. AI技术 ⭐ （核心）
- 新模型发布、API 变更、框架版本更新
- RAG/Agent 架构演进
- Prompt Engineering 最佳实践
- AI 产品设计趋势、多模态应用落地

### 2. 数码硬件
- 最新硬件参数、评测、操作系统更新
- AI PC/NPU 芯片进展
- 智能穿戴设备
- 生产力工具硬件（鼠标/键盘/显示器/音频）

### 3. 时事政治
- 国际地缘格局、中美科技博弈
- AI 监管政策、数据安全与隐私法规
- 互联网行业政策动向

### 4. 文学
- 写作方法论（叙事结构、人物塑造）
- 文学批评与书评
- 经典重读

### 5. 历史
- 中国近现代史、明清政治史
- 科技发展史、商业/企业史
- 历史人物传记与权谋分析

### 6. 个人提升
- 时间管理与效率系统
- 认知心理学
- 沟通与表达、职业发展规划
- 第二大脑/知识管理方法论

### 7. 游戏 🎮
- **即时战略 (RTS)**：星际争霸、帝国时代、红警 — 资源调度、多线操作、战略博弈
- **角色扮演 (RPG)**：技能树规划、装备搭配、剧情分支选择
- **Roguelike/策略**：风险评估、Build 构建、随机应变

## 类比库（供 Agent 讲解时使用）

| 兴趣领域       | 可类比到           | 示例                                |
| ---------- | -------------- | --------------------------------- |
| AI技术/Agent | 团队协作、外包管理      | "Agent 权限就像给实习生开通系统权限"            |
| 历史/权谋      | 复杂系统博弈、策略选择    | "这个架构决策就像诸葛亮的隆中对"                 |
| 文学/叙事      | 内容结构、信息呈现      | "这段代码的逻辑就像小说的三幕结构"                |
| 效率系统       | 工作流设计、自动化      | "这个流程就像你的第二大脑收件箱"                 |
| 硬件评测       | 性能权衡、参数对比      | "模型选择就像选显卡，看你是跑什么任务"              |
| RTS/即时战略   | 资源调度、多任务并行、微操  | "学习路线规划就像开局运营，先攒经济还是Rush"         |
| RPG/角色扮演   | 能力树规划、Build 构建 | "知识体系就像技能树，点亮前置才能解锁后续"            |
| Roguelike  | 风险评估、随机应变      | "这个概念有陷阱，就像Roguelike里的精英怪，不小心就暴毙" |
